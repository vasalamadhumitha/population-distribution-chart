# Population Distribution Chart

This project uses Python and Matplotlib to visualize population distribution
across different categories using a bar chart.

## Objective
To visualize population distribution using Python and Matplotlib.

## Files Included
- chart.py – Python script to generate the chart
- chart.png – Output image of the population distribution chart
- population distribution chart.txt – Description of the chart

## Requirements
- Python 3.x
- matplotlib

## How to Run
```bash
python chart.py
